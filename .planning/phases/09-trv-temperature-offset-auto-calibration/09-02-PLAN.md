_async_calibrate(self, config: dict) -> None (D-01/D-02/D-03/D-04, mirrors the
push-pass gather at coordinator.py lines 256-274):
  - First line: if not config.get("calibration_enabled", False): return (D-04 —
    no service calls, no attribute reads when disabled)
  - rooms = self._data.rooms
  - await asyncio.gather(*(self._async_calibrate_room(area_id, entity_id, config)
    for area_id, entity_ids in rooms.items() for entity_id in entity_ids if
    is_trv_entity(self._hass, entity_id)))

_async_calibrate_room(self, area_id, entity_id, config) -> None — implement the
full skeleton from 09-RESEARCH.md "Code Examples" section, in this guard order
(every guard is a silent return, no log, no raise):
  1. room_config = config.get("rooms", {}).get(area_id, {}); sensor_entity_id =
     room_config.get("temperature_sensor"); if not sensor_entity_id: return
     (CALIB-05, D-14 — only the manually configured key; never read
     room_auto_sensors or area registry, per Anti-Patterns)
  2. if not supports_offset_calibration(self._hass, entity_id): return
     (CALIB-03, D-08)
  3. state = self._hass.states.get(entity_id); if state is None or state.state
     == "unavailable": return (ROOM-03 parity)
  4. sensor_state = self._hass.states.get(sensor_entity_id); if sensor_state is
     None or sensor_state.state in ("unavailable", "unknown"): return; then
     float() cast wrapped in try/except (ValueError, TypeError): return
     (Pitfall 5, mirrors _build_status_payload guard at lines 375-384)
  5. current_temp = state.attributes.get("current_temperature"); if current_temp
     is None: return; then float() cast wrapped in try/except: return (Pitfall 2)
  6. delta = sensor_temp - current_temp (D-05)
  7. threshold = config.get("calibration_threshold", 0.5); if abs(delta) <=
     threshold: return (D-07, CALIB-04)
  8. existing_offset = state.attributes.get("temperature_offset", 0.0), float()
     cast with try/except defaulting to 0.0; new_offset = existing_offset +
     delta (D-06 — read temperature_offset attribute, NOT current_temperature;
     Pitfall 1)
  9. try: await set_trv_offset(self._hass, entity_id, new_offset) except
     Exception:  # noqa: BLE001 -> _LOGGER.warning("Failed to apply offset %.1f
     to %s", new_offset, entity_id) (Pitfall 3 / _push_safely no-propagate
     parity)

Call site (D-01): add `await self._async_calibrate(config)` at the very END of
async_evaluate, AFTER the self._hass.bus.async_fire(...) status block (currently
