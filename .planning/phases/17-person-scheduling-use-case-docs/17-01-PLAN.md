use-case folders depend on.

Makefile per use-case folder orchestrated by an append-only root delegation
target that leaves the existing docs/screenshots/ panel-tab set untouched.

scenario-mode capture branch) and a root `Makefile` with a
`use-case-screenshots` target that `screenshots` invokes after its existing
docker run.
