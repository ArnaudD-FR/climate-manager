# Quick Task 260526-k71: Reset-to-global delegates to new backend WS command

## One-liner

New `climate_manager/reset_room_to_global_program` WS command owns the global→room schedule deep-copy; `_onResetToGlobal` reduced to a dumb send-and-reload relay.

## What Was Built

**Task 1 — Backend WS command + pytest**

Added `_make_ws_reset_room_to_global_program` factory to `websocket.py`:
- Schema: `{type: "climate_manager/reset_room_to_global_program", room_id: str}`
- Handler deep-copies `runtime_config.global_time_program` into `rooms[room_id].time_program` and sets `rooms[room_id].room_mode = "custom"`
- Uses `copy.deepcopy` so room and global never share list references (T-03-09)
- Sparse-merge via `setdefault` — sibling rooms and all other top-level keys untouched
- `write-then-evaluate` pattern: `store.async_save` → `send_result` → `coordinator.async_evaluate` (background task)
- Registered in `async_register_commands` after `_make_ws_reset_time_program`
- Module docstring updated from "10 WebSocket commands" to "11", with new bullet entry

New test `test_ws_reset_room_to_global_program_copies_global_into_room` verifies:
- Result `success: True`
- Target room `room_mode == "custom"`
- Target room `time_program` deep-equals `global_time_program`
- Deep-copy proof: mutating target room's time_program does NOT bleed into `global_time_program`
- Sibling room (`room-b`) completely untouched

**Task 2 — Frontend ws-client + room-card + bundle rebuild**

`frontend/src/ws-client.ts`:
- Added `resetRoomToGlobalProgram(roomId: string): Promise<{ success: boolean }>` immediately after `resetTimeProgram`, using `sendMessagePromise` with `type: "climate_manager/reset_room_to_global_program"` and `room_id: roomId`

`frontend/src/components/room-card.ts`:
- `_onResetToGlobal` replaced from 13-line body (with `JSON.parse/JSON.stringify` and `global_time_program` access) to a 7-line dumb relay: `this.ws.resetRoomToGlobalProgram(this.roomId)` → `panel.reloadConfig()` → toast
- `DailyProgram` import retained (still used by `_lastTimeProgram` field, `_onRoomModeChange`, and `_onPeriodsChanged`)

`www/panel.js`:
- Rebuilt with Vite; bundle contains `reset_room_to_global_program` string
