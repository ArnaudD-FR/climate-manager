# Quick Task 260520-s2t: Room Card Content Top Padding

## Root Cause

`.card-content` in `room-card.ts` has `padding: 0 16px 16px` — zero top padding
— so the "CLIMATE ENTITIES" label sits flush against the divider line separating
it from the header.

## Fix

Change `.card-content` padding from `0 16px 16px` to `12px 16px 16px` in
`frontend/src/components/room-card.ts`.
