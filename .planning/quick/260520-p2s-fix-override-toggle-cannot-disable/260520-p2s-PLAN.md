# Quick Task 260520-p2s: Fix Override Toggle Cannot Disable

## Root Cause

In `room-card.ts` `_onOverrideToggle`, when disabling the override:

```typescript
timeProgram = null;
await this.ws.setRoomConfig(this.roomId, { time_program: timeProgram ?? undefined });
```

`null ?? undefined` evaluates to `undefined`. JSON serialization drops `undefined` values from
objects, so the backend receives `config: {}`. The backend's `.update({})` is a no-op — the
`time_program` key is never cleared from storage.

## Fix

1. `frontend/src/types.ts` — allow `null` on `RoomConfig.time_program`:
   ```typescript
   time_program?: DailyProgram | null;
   ```

2. `frontend/src/components/room-card.ts` — remove `?? undefined`, send `null` directly:
   ```typescript
   await this.ws.setRoomConfig(this.roomId, { time_program: timeProgram });
   ```

## Files

- `frontend/src/types.ts`
