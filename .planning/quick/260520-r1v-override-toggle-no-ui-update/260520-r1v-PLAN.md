# Quick Task 260520-r1v: Override Toggle No UI Update

## Root Cause

`_onOverrideToggle` and `_onPeriodsChanged` in room-card.ts call `ws.setRoomConfig`
but never call `panel.reloadConfig()`. The parent's `_config` stays stale, so
`this.config` never updates — `hasCustomProgram` keeps its old value, the badge
and time-bar don't re-render, and the status line doesn't reflect the new state.

## Fix

Add `await this.panel.reloadConfig()` after the successful save in both
`_onOverrideToggle` and `_onPeriodsChanged` in `frontend/src/components/room-card.ts`.
