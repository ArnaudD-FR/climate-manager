# Sofia — Shared Custody (Odd/Even Weeks)

Sofia is a child present at home on even weeks and away with the other parent on
odd weeks. This example shows how to use the **scheduled even/odd weeks**
presence mode, where two independent weekly schedules alternate on a
per-ISO-week basis.

## Configuration Summary

| Property       | Value                                                  |
| -------------- | ------------------------------------------------------ |
| Presence mode  | `scheduled` (even/odd weeks)                           |
| Schedule shape | Even week: present all week; odd week: absent all week |
| Assigned rooms | Bedroom                                                |
