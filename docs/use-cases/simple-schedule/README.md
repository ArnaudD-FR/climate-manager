# Emma — Simple Schedule

Emma works a standard weekday job and follows a regular routine: away from home
during office hours, present mornings and evenings, home all weekend. This
example shows how to use the **scheduled (single week)** presence mode to
capture that pattern.

## Configuration Summary

The table below is the Configuration Summary for this use case.

| Property       | Value                                                         |
| -------------- | ------------------------------------------------------------- |
| Presence mode  | `scheduled` (single week)                                     |
| Schedule shape | Away weekdays ~09:00–17:30, home mornings/evenings + weekends |
| Assigned rooms | Bedroom, Living Room                                          |
